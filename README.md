# Saude Já

    Este projeto visa ajudar pessoas dentro de uma cidade a conseguirem encontrar os diversos preços de um remédio dentro de suas cidades de forma mais fácil e rápida, compilando o preço de um mesmo remédio em diversas lojas e listando-os para os usuários em uma única tela. 

    O Saude Já além disso, tem a ideia de promover a reciclagem de frascos de remédios que ainda podem ser reutilizados, assim, reduzindo lixos, por mostrar quais lojas na cidade recebem reciclagem.

    Por fim, o Saúde Já conta com uma tela onde mostra médicos cadastrados no sistema e demonstra suas informações de contato, promovendo assim, uma maior aproximação do usuário com os médicos.
